**Project Overview**
This repository contains a practice data analytics project developed using Power BI to analyze sales performance and outlet characteristics for Blinkit, an online grocery delivery platform.

The dashboard demonstrates how business data can be transformed into interactive visual insights to understand sales trends, product performance, outlet distribution, and customer ratings.

This project was created for data visualization and analytics practice, focusing on building an intuitive dashboard that highlights key business metrics.

**Objective**

The objective of this project is to practice:

Building interactive dashboards in Power BI

Analyzing retail and grocery sales data

Understanding product category performance

Evaluating outlet characteristics and sales contribution

Presenting business insights through clear visual storytelling

**Tools & Technologies Used**

Power BI – Data visualization and dashboard creation

Microsoft Excel – Data source

DAX (Data Analysis Expressions) – Calculated measures

**Key Insights**

From the dashboard analysis, several observations can be made:

Tier 3 locations contribute the highest sales, indicating strong demand in those regions.

Supermarket Type 1 outlets generate the highest overall revenue.

Fruits & Vegetables and Snack Foods are among the top-selling item categories.

Regular fat products contribute slightly more revenue than low-fat items.

Average customer ratings remain consistently around 4, indicating stable customer satisfaction.
